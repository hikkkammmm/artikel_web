# CI-BLOG: Portal Berita Teknologi

Sebuah aplikasi portal berita sederhana yang dibangun menggunakan **CodeIgniter 4**. Proyek ini menggunakan arsitektur MVC (Model-View-Controller) dan memanfaatkan **Session** untuk menyimpan data artikel secara sementara (tanpa menggunakan database).

## 🚀 Fitur Utama
* Menampilkan daftar artikel berita dengan tampilan modern (Grid & Card).
* Halaman detail artikel yang dinamis.
* Form penambahan artikel baru (Data disimpan sementara di *Browser Session*).
* UI/UX responsif menggunakan Bootstrap.

---

## 📋 Persyaratan Sistem (Prerequisites)
Sebelum menjalankan proyek ini, pastikan komputer/laptop Anda sudah terinstall:
* **PHP** (Minimal versi 7.4, direkomendasikan versi 8.1 atau lebih baru)
* **Composer** (Package Manager untuk PHP)
* Web Server lokal (XAMPP, Laragon, dll) - *Opsional jika menggunakan Spark CLI*
